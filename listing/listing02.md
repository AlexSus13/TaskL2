Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
fmt.Println(test()) вернет 2.
fmt.Println(anotherTest()) вернет 1.
defer функции вызываются непосредственно перед возвратом из окружающей функции в порядке, обратном их отложенному выполнению.
defer вызывается после того, как значение в return уже установлено.
Единственный способ для defer функции получить доступ к возвращаемому результату — использовать именованный параметр результата.
Функция test() возвращает  именованный параметр x ==> перед возвратом x изменяется.
Функция anotherTest() возвращает  неименованный параметр ==> перед возвратом x не изменяется.
```
